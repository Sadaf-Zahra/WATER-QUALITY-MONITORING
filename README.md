*Water Quality Monitor* is an integrated environmental monitoring system designed to address the challenge of accessible water safety by providing real-time potability analysis and automated surface waste classification. By combining chemical sensors with Deep Learning-based Computer Vision, the project identifies chemical imbalances and floating debris through an automated hardware suite and camera feed. This innovation serves as a low-cost, portable alternative to traditional laboratory testing, enabling field researchers to navigate complex environmental data while addressing the immediate need for safe drinking water through personalized, AI-driven analysis.

## Technical Approach
  * Phase I (Research): Benchmarked high-dimensional models (XGBoost, CatBoost, TabNet) and deep learning architectures (ResNet50, DenseNet121) to establish performance baselines.

  * Phase II (Implementation): Optimized findings into a field-deployable MVP using an Arduino sensor suite (pH, TDS, Turbidity) and a Python-based dashboard for live visualization and CSV data logging.
